# Caso de Estudio: Cobertura de Vacunación DTP3 (2020–2024) – Salud Pública Global

El proyecto presenta un análisis de datos masivos sobre la cobertura de vacunación DTP3 en niños de 12 a 23 meses a nivel global, destacando su importancia en la salud pública y el impacto de la pandemia de COVID-19.
Selección de Temática sobre Vacunación

Se estudia la cobertura de vacunación DTP3 en niños de 12-23 meses.
La vacunación es clave para la salud pública y el desarrollo sanitario (ODS 3).

# Fuentes de Recolección de Datos

Se utilizan datos de Our World in Data (OWID) y del Banco Mundial.
OWID ofrece datos granulares desde 1980, mientras que el Banco Mundial proporciona datos agregados para 2020-2024.

# Métodos y Herramientas de Análisis

Se descargaron datos en formatos CSV y JSON.
Se asegura la calidad de los datos mediante limpieza y preparación.

# Almacenamiento de Datos en PostgreSQL

Se eligió una base de datos relacional para garantizar integridad referencial y realizar consultas analíticas complejas.
PostgreSQL es robusto y adecuado para datos estructurados.

# Análisis Prescriptivo y Predictivo

Se seleccionaron variables clave para el análisis descriptivo y se aplicó un modelo de regresión lineal simple.
El modelo busca predecir la cobertura DPT a nivel global.

# KPIs y Recomendaciones para el Dashboard

Se establecieron KPIs para monitorear la cobertura de vacunación y alertar sobre riesgos.
Se sugieren intervenciones basadas en los resultados de los KPIs analizados.

# Conclusiones y Limitaciones del Estudio

El análisis ayuda a identificar tendencias y priorizar regiones con baja cobertura.
Limitaciones incluyen un marco temporal corto y posibles sesgos en estimaciones.
