
# Descubrimiento de patrones y agrupamiento en aperturas de ajedrez 

El proyecto presenta un análisis de patrones y agrupamientos en aperturas de ajedrez utilizando técnicas de minería de datos.

# Objetivo del Problema

Aplicar técnicas descriptivas de minería de datos para identificar patrones y asociaciones en aperturas de ajedrez.
Analizar porcentajes de victorias, empates y fracasos según el tipo de apertura utilizada.

# Descripción del Dataset

El dataset, obtenido de Kaggle, contiene 1882 instancias y 26 atributos, incluyendo el código ECO, nombre de la apertura, movimientos y variaciones.
El sistema ECO clasifica aperturas de ajedrez mediante códigos alfanuméricos, facilitando su identificación y estudio.

# Análisis de Datos con K-Means

Se utilizó el método del codo para determinar la cantidad óptima de clusters, revelando patrones en los porcentajes de victorias y empates según el tipo de jugador.
Los resultados muestran variaciones significativas en el rendimiento de las aperturas, destacando la supremacía de las blancas en algunos clusters y el aumento de victorias para las negras en otros.

# Conclusiones Finales

El aprendizaje no supervisado permite simplificar un conjunto complejo de aperturas en grupos representativos.
Se sugiere que jugadores noveles se enfoquen en los clusters principales para su estudio.
La metodología aplicada demuestra ser efectiva para revelar estructuras ocultas en el análisis de aperturas de ajedrez, con implicaciones más amplias en otros dominios.
